## üöÄ How to Run

These are standard Java console applications.

1. **Clone the repo**:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. **Navigate** to a specific block folder.
3. **Compile** using `javac`:
   ```bash
   javac Main.java
   ```
4. **Run** the class file:
   ```bash
   java Main
   ```
   *(Note: Some IDE-specific folders like `.idea` are present if you prefer opening them in IntelliJ IDEA).*
